# 快速开始

## 1. 配置插件

在 Dify 中导入插件后，配置以下参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| ARK API Key | 必填 | `your-api-key` |
| Base URL | 可选，默认已设置 | `https://ark.cn-beijing.volces.com` |
| Model | 可选，模型接入点（接入点） | `doubao-seedance-1-5-pro-251215` |

## 2. 使用工具

### 文生视频

```
提示词: "一只可爱的猫咪在草地上玩耍"
时长: 5秒
宽高比: 自适应
生成音频: 是
```

### 图生视频

```
参考图片: 上传1张图片 (本地文件或URL)
提示词: "[图1]的角色开始跳舞"
时长: 5秒
宽高比: 16:9
生成音频: 是
```

## 3. 等待结果

视频生成是异步的，插件会自动：
1. 创建任务
2. 轮询任务状态 (每5秒查询一次，最多120次)
3. 任务完成后返回视频文件和详细信息

## 注意事项

- 提示词建议: 中文不超过500字，英文不超过1000词
- 参考图片要求: 格式为jpeg/png/webp/bmp/tiff/gif，大小小于30MB
- 视频时长: 1-10秒
- 当前工具配置为单张参考图片输入
