# Privacy Policy for Doubao Ark Video Plugin

This plugin is a Dify custom plugin that interfaces with Doubao Ark's video generation API.

## Data Processing

This plugin requires the following credentials to operate:
- **ARK API Key**: Used for authenticating requests to the Doubao Ark API.
- **Base URL**: The endpoint for the Doubao Ark API.
- **Model**: The specific model/access point to use for video generation.

These credentials are stored securely within Dify's plugin credential management system.

## API Requests

When you use this plugin to generate videos, it makes API requests to Doubao Ark's servers. The data sent includes:
- Your prompt text (for text-to-video)
- Reference images (for image-to-video, either as URLs or base64-encoded)
- Video generation parameters (duration, aspect ratio, etc.)

All API requests are made directly from your Dify instance to Doubao Ark's API endpoints. The video generation processing happens on Doubao Ark's servers according to their privacy policy.

## File Handling

For image-to-video generation with local file uploads:
- Uploaded images are temporarily stored in Dify's file manager
- Images are converted to base64 and sent to the Doubao Ark API
- No persistent storage of uploaded files occurs within the plugin itself

## Recommendations

- Always use HTTPS endpoints to ensure secure communication
- Keep your ARK API key confidential and do not share it publicly
- Review Doubao Ark's terms of service and privacy policy before using this plugin

This plugin is provided as-is. The plugin author is not responsible for the content generated by the Doubao Ark API or any data handling practices of Doubao Ark.
